формат файла KONTR.DBF

YNN      C(20)  -- УНП партнера
NAME     C(50)  -- Краткое наименование партнера (служебное поле)
FULLNAME C(200) -- Официальное наименование партнера
OKPOCOD  C(20)  -- ОКПО
LICENSE  C(20)  -- лицензия (пока не заполняется)
ADRESS   C(20)  -- адрес партнера
PHONE    C(20)  -- телефон
RS       C(20)  -- расчетный счет
BANK     N(5,0) -- код банка, 3-х значный




формат файла Documents.dbf

ID         N(10,0) -- внутренний уникальный идентификатор документа
TYPE       N(5,0)  -- тип движения 1-приход, 2-расход, 3-переоценка или инвентаризация (итоговые разницы)
KIND       N(10,0) -- идентификатор типа документа из справочника <типовые документы> (описаны ниже)
DOCDATE    D(8)    -- дата документа
NUMBER     C(30)   -- номер документа
UNN        C(20)   -- УНП предприятия, если документ предполагает стороннего партнера
SUMF       N(19,4) -- сумма по документу в ценах изготовителя (для 1С это не требуется)
SUMN       N(19,4) -- сумма по документу в ценах поставщика (покупная или отпускная стоимость)
SUMPALL    N(19,4) -- сумма по документу в розничных ценах
в том числе
SUMP_0   N(19,4) -- сумма облагаемая НДС поставщика по ставке 0%
SUMP_10  N(19,4) -- сумма облагаемая НДС поставщика по ставке 10%
SUMP_18  N(19,4) -- сумма облагаемая НДС поставщика по ставке 20%

NDSIN    N(19,4) -- сумма НДС поставщика
в том числе:
NDSIN_10 N(19,4) -- сумма НДС поставщика по ставке 10%
NDSIN_18 N (19,4) -- сумма НДС поставщика по ставке 20%


NDSOUT     N(19,4) -- Сумма розничного НДС
в том числе
NDSOUT_10 N(19,4) -- Сумма розничного НДС по ставке 10%
NDSOUT_18 N(19,4) -- Сумма розничного НДС по ставке 20%


TRADERAISE N(19,4) -- Сумма торговой наценки
NP         N(19,4) -- Сумма налога с продаж
ISFIXPRICE N(5,0)  -- тип расчета цены: 0-обычный, 1- по фиксированным ценам (это вряд ли понадобиться)
ISTARA     N(5,0)  -- 0 означает товар, 1-тара
ROUNDOFF   N(19,4) -- сумма округления
GLASSPRICE N(19,4) -- транспортные расходы
GLASSNDS   N(19,4) -- НДС транспортных расходов
DESCRIBE   C(50)   -- свободное для примечаний поле

для учета реализации товара через кассы при оплате банковскими карточками добавлено два поля
SUMNAL     N(19,4) -- сумма оплаты по док-ту реализации нал. деньгами
SUMBNAL    N(19,4) -- сумма оплаты по док-ту реализации безналичными деньгами

для остальных типов док-тов эти поля нулевые

SUMMWITHD N(19,4) -- сумма товара со скидкой
SUMMD N(19,4) -- сумма сумма скидки
SUMMTRD N(19,4) -- сумма скидки за счет торговой надбавки
SUMMNDSD N(19,4) -- сумма скидки за счет НДС
SUMMROUNDD N(19,4) -- сумма скидки за счет округления


справочник типовых документов
 2-инвентаризация 
 3 -- внутренее перемещение
 4-реализация товара через кассы 
 5-ТТН на приход товара от поставщика
 8-ТТН на возврат товара поставщику
 7-реализация за безналичный расчет (юр. лицам)
98-возврат товара покупателем с выплатой денег
99-переоценка
